# BotHost - Aloja tus Bots Fácilmente

Plataforma web para alojar bots de Telegram, Discord, WhatsApp y más.

![Demo](https://via.placeholder.com/800x400?text=BotHost+Demo)

## ✨ Características
- Registro e inicio de sesión
- Dashboard moderno y colorido
- Despliegue de bots en 1 clic
- Consola en vivo
- Estadísticas en tiempo real

## 🚀 Cómo probar
1. Ve a: `https://tunombre.github.io/bot-host`
2. Regístrate con cualquier usuario y contraseña
3. ¡Despliega tus bots!
